# 🏫 大甲國中教務處 - 活動報名與繳費查詢系統

這是一套專為國中教務處打造的**現代化雲端報名與核帳系統**。
透過「前後端分離」架構，提供家長流暢的行動端查詢體驗，同時大幅減輕行政人員的收費對帳負擔。

## 🏗️ 系統架構 (Architecture)

本系統採用輕量化、零伺服器成本的 Serverless 架構：
* **前端介面 (Frontend):** 託管於 **GitHub Pages**。採用 HTML5 + Bootstrap 5，結合 Canva 風格的現代化 UI 設計 (Glassmorphism 玻璃透視感)。
* **後端大腦 (API Gateway):** 部署於 **Google Apps Script (GAS)**。負責接收 RESTful API 請求、處理業務邏輯與產出 PDF。
* **資料庫 (Database):** 介接 **Google 試算表 (Google Sheets)**。提供教務處最熟悉的備份與檢視介面。

---

## ✨ 核心功能介紹 (Features)

### 👨‍👩‍👦 家長 / 學生查詢端 (`index.html`)
* **直覺式防呆查詢**：輸入正確的「年、班、號、姓名」(具備防負數機制) 即可查詢，保護學生個資。
* **即時狀態追蹤**：一目瞭然的「待繳費、已繳費、異常退件」狀態顯示。
* **動態雙聯單 PDF 下載**：
    * **繳費前**：顯示「應繳金額」，作為家長簽名的報名同意書與繳費通知。
    * **繳費後**：自動蓋上「已繳費收款章」，顯示「實收金額」與列印時間，具備正式收據效力。

### 💼 教務處管理後台 (`admin.html`)
系統化整合教務處 5 大日常作業需求：
1.  **單筆錄入**：支援輸入座號自動帶出「學生名冊」姓名，自動計算「報名級別」與「加購項目」總額。
2.  **📂 整批匯入**：支援直接從 Excel 複製貼上「年 班 號」欄位，系統自動比對名冊並批次建檔，數百筆資料一秒寫入。
3.  **快速核帳**：
    * 支援關鍵字 (姓名/收據編號) 搜尋，或一鍵「顯示全部」名單。
    * 可快速修改「實收金額」與「繳費狀態」，無縫應對特殊減免或補繳狀況。
    * 提供「補印雙聯單據」功能，解決學生遺失單據的痛點。
4.  **活動概覽 (Dashboard)**：首頁即時顯示各項活動的「報名人數」、「已收總額」、「剩餘天數」，並支援一鍵 **匯出 CSV** 供後續造冊使用。
5.  **系統設定**：全圖形化介面建立新活動，可自訂比賽日期、費用級別與加購項目，無需動手修改程式碼。

## 🚀 部署與維護指南 (Deployment Guide)

### 前端部署 (GitHub)
1.  將 `index.html` 與 `admin.html` 放置於 Repository 根目錄。
2.  啟用 GitHub Pages 服務，設定分支為 `main`。
3.  **重要：** 必須確認 HTML 檔案中的 `const API_URL` 已正確填入最新版的 GAS Web App 網址。

### 後端更新 (Google Apps Script)
1.  所有業務邏輯均撰寫於 `Code.gs`。
2.  若有修改 PDF 版面或 API 路由，請務必點選 **「部署 > 管理部署 > 建立新版本」** 才會生效。

### 資料庫維護 (Google Sheets)
* 系統依賴 `Config_Settings` 讀取活動選項。
* 系統依賴 `學生名冊` 進行姓名防呆驗證。每學年開學請務必更新此工作表的內容，以確保查驗功能正常運作。
